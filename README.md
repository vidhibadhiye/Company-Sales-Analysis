Sales Analysis Dashboard

This project is a simple and interactive dashboard to analyze company sales data. It helps understand trends like which products sell the most, which regions perform better, and how sales change over time.

 What it does
Shows monthly sales trends 
Highlights best-selling products 
Displays category-wise distribution 
Compares region-wise revenue 
Lets you upload your own CSV file and see results instantly
Built With
Python (Pandas, NumPy, Matplotlib)
HTML, CSS, JavaScript
Chart.js
 How to use

Just open the HTML file in your browser and upload a CSV file. No setup needed.
